md2files module

This script converts Markdown files to various output formats using Pandoc. Currently supported output formats are: PDF, HTML, LaTeX, and JATS. The container expects a yaml file with metadata and a markdown file Optionally, a BibTeX bibliography and individual Lua filters can be passed.
